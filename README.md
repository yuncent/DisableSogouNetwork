# DisableSogouNetwork - Sogou Input Method Network Blocker (Windows Firewall Tool / 禁止搜狗输入法联网工具)

Block Sogou Input Method from accessing the internet using Windows Firewall rules (disable sogou network / 禁止搜狗输入法联网).

一款用于 Windows 的工具，可通过防火墙规则彻底禁止搜狗输入法联网，阻止其网络通信与数据上传。

---

## 🎯 Features / 功能

- Scan Sogou Input Method processes (搜狗输入法进程扫描)
- Detect installed and running Sogou components
- View firewall status (enabled / blocked)
- One-click block network access (disable sogou network)
- One-click restore network access
- Remove leftover firewall rules
- Support custom scan folders
- Detect hidden or residual Sogou rules

## ⚙️ Usage / 使用方法

1. Run as Administrator (以管理员身份运行)
2. Click "Scan Processes"（扫描进程）
3. Select Sogou Input Method processes
4. Click "Block Network" or "Restore Network"

---

## 🧠 Description / 项目说明

This tool blocks all network access of Sogou Input Method using Windows Firewall rules to prevent data transmission and improve privacy.

本工具通过 Windows 防火墙规则，彻底禁止搜狗输入法联网，从而阻止其后台网络通信，提升系统隐私安全。
